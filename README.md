## ✨ Features
*   **Full CRUD Operations:** Create, Read, Update, and Delete notes seamlessly.
*   **Persistent Storage:** Uses `localStorage` to save notes so data isn't lost on refresh.
*   **Search & Filter:** Quick search bar to filter notes by title or content instantly.
  

---

## 🛠️ Tech Stack
*   **Frontend:** React.js / Vanilla JavaScript
*   **Storage:** LocalStorage API
*   **Icons:** FontAwesome
